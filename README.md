## 🗄️ BASE DE DATOS SQLITE

El sistema utiliza **SQLite** como almacenamiento unificado (`utils/database.py`). La base de datos se encuentra en `data/geoint.db` y se crea automáticamente al iniciar la aplicación.

### Tablas

| Tabla | Propósito |
|-------|-----------|
| `analysis_history` | Historial completo de análisis con prompt, respuesta, modelo, región, país |
| `article_cache` | Caché de artículos OSINT (TTL 2h) por país y categoría |
| `reports` | Registro de informes generados (ruta, nombre, países, clasificación, tamaño) |
| `schedules` | Tareas programadas de análisis (persistidas entre reinicios) |

### Características

- **Thread-safe**: conexiones protegidas con `threading.Lock`
- **WAL mode**: journal WAL para mejor rendimiento en lecturas concurrentes
- **Migraciones automáticas**: schema versionado con `PRAGMA user_version`
- **Caché unificada**: reemplaza el antiguo sistema de archivos JSON en `.cache/`
- **RAG**: función `build_context_prompt()` para contexto de análisis previos

### Ciclo de vida de la caché

Los artículos OSINT se cachean 2 horas en `article_cache`. Si hay suficientes artículos frescos para un país, se evita una nueva recolección. La tabla se limpia automáticamente al insertar nuevos artículos.

### Cómo añadir un canal de YouTube

Para añadir canales de YouTube a `sources/sources.json`:

1. Ve al canal de YouTube
2. La URL tendrá el formato: `https://www.youtube.com/channel/UC...`
3. Copia el `channel_id` (la parte después de `/channel/`)
4. Añade la entrada en `youtube_channels` con nombre, ID, país, sesgo y fiabilidad

   P: ¿Es gratis?

   P: ¿Necesito internet?
   R: Solo para descargar las noticias. La IA funciona sin internet.

   P: ¿Mis datos se envían a algún sitio?
   R: NO. Todo se queda en tu ordenador. Nada sale de tu PC.

   P: ¿Puedo usarlo en un ordenador sin internet (air-gapped)?
   R: Sí. Una vez instalado Ollama y el modelo, la IA funciona

      las fuentes de noticias.

   P: ¿Cuánto tarda en generar un informe completo?
   R: Con gemma3:4b y sin GPU: ~30-60 minutos para 23 países.
      Con GPU NVIDIA: ~10-15 minutos.

   P: ¿Puedo añadir más países?
   R: Sí. Edita config.yaml y añade el país con su código ISO.

   P: ¿Puedo añadir más fuentes de noticias?
   R: Sí. Edita sources/sources.json y añade feeds RSS o canales
      de YouTube.

   P: ¿Los análisis son fiables?
   R: La IA analiza información de fuentes públicas. Siempre debe
      validarse por un analista humano antes de uso operativo.
      La IA NO inventa noticias, solo analiza las que lee.
